# GHZ-quantum_computing
A code For experiencing GHZ (Greenberger–Horne–Zeilinger) entangled quantum state that involves at least three qubits are involved.

## 🚀 How to Run
1. Clone the repo
2. Install dependencies: `pip install qiskit qiskit-ibm-runtime matplotlib`
3. Set up your IBM Token (Use environment variables for security!)
4. Run the script
